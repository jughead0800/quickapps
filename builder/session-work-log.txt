7. CREATED GITEA ACTIONS WORKFLOW
   - Created .gitea/workflows/build.yml
   - Automatically builds index.html when:
     * Files in apps/ folder are modified
     * build-index.sh is updated
     * build-index.ps1 is updated
     * Workflow file itself is updated
   - Uses actions/checkout@v3 (same pattern as working PowerShell build)
   - Runs on ubuntu-latest (no container needed)
   - Includes skip CI logic ([skip ci] in commit messages)
   - Only commits on push to main (not PRs)
   - Uses permissions: contents: write for push access
   - File: .gitea/workflows/build.yml

8. WORKFLOW TROUBLESHOOTING
